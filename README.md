## 🚀 설치 및 실행 방법

1. 저장소 복제

```bash
git clone https://github.com/your-username/portfolio.git
cd portfolio
```

2. 의존성 설치

```
bash
Copy
npm install
# or
yarn install
# or
pnpm install
```

3. 개발 서버 실행

```
bash
Copy
npm run dev
# or
yarn dev
# or
pnpm dev
```
